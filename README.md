# NeuraMed 🏥🤖

NeuraMed is an AI-powered healthcare platform that unifies advanced Machine Learning models, Optical Character Recognition (OCR), Voice Interaction, and smart analytics into a single comprehensive patient tracking and medicine management system. The platform consists of a robust Next.js frontend and a highly modular FastAPI gateway orchestrating multiple intelligent microservices.

## 🌟 Key Features

- Prescription OCR & Parsing: Upload images of medical prescriptions and instantly extract structured dosage, duration, and medicine names using Google Vision and GPT-4o.
- Voice System (Sarvam AI Integration): Supports regional spoken languages (e.g., Hindi/English via Sarvam API) and handles Text-to-Speech (TTS) / Speech-to-Text (STT) for fully localized AI-to-patient interactions.
- AI Safety & Drug Interactions: Utilizes a combination of rule-based logic and Machine Learning models (interaction_model.pkl) to predict and warn about potentially unsafe drug combinations in real-time.
- Smart Patient Dashboard & Database: Persistent tracking of prescriptions, medicine adherence logs, and patient records using Firebase and SQLAlchemy.
- Automated Alerts & Reminders: Configurable alert systems designed to automatically nudge patients about upcoming doses via voice or text notifications.
- Responsive Web Interfaces: Built with Next.js 15, React 19, Tailwind CSS, Framer Motion, and Zustand for state management. Features smooth animations and a medical-grade visual theme.

## 🛠 Tech Stack

### Frontend
- Framework: [Next.js 15](https://nextjs.org/) (App Router)
- Library: React 19, Lucide React (Icons)
- State Management: Zustand
- Animations: Framer Motion
- Styling: Tailwind CSS

### Backend & AI Orchestration
- API Gateway: FastAPI (Python)
- Database: SQLAlchemy (ORM) / Firebase
- AI & NLP: 
  - OpenAI GPT-4o (Intent Extraction & Validation)
  - Google Cloud Vision (Image OCR)
  - Sarvam AI (Regional Speech Processing)
  - Scikit-learn (Custom ML Models for Adherence prediction and Drug interactions)

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- Python 3.9+
- Provide API Keys for OpenAI, Google Cloud, and Sarvam in .env based on .env.example

### 1. Setup Backend
# Go to project root Directory
cd Neura-Med

# Install Python dependencies
pip install -r requirements.txt

# Run the unified gateway (FastAPI) locally
python main.py
# The API will be available at http://localhost:8000

### 2. Setup Frontend
cd frontend

# Install Node.js dependencies
npm install

# Start Next.js development server
npm run dev
# The frontend app will be available at http://localhost:3000
